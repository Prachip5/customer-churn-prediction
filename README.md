# Customer Churn Prediction using Machine Learning

## Project Overview
This project aims to predict customer churn using machine learning techniques. Customer churn refers to customers who stop using a company's services. By identifying such customers early, businesses can improve retention strategies.

## Problem Statement
The objective is to build a predictive model that classifies whether a customer is likely to churn based on historical customer information.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Preprocess customer data
- Build machine learning classification models
- Evaluate model performance
- Understand the complete ML workflow

## Dataset Information
Dataset: Telco Customer Churn Dataset

Approximate Dataset Size:
- Around 7,000 customer records
- Approximately 21 features
- Size: ~1–2 MB
